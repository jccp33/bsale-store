Objective

Detail the development of this project, publish the endpoints and the methods and data necessary to access them.

BackEnd

The BackEnd was developed entirely with PHP without any framework or library. Being a "test" project, some aspects were not considered for this development, such as endpoint authentication operations.

FrontEnd

                

                    The FrontEnd was developed entirely with HTML, CSS and JavaScript without any framework or library. Requests to the server are handled asynchronously with an AJAX function.
